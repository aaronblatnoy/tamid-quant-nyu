TAMID QUANT — LOGO ASSET LIBRARY
=================================

01_mark/
  Mark only. The hollow chamfered T+G symbol, no wordmark.
  Use when context already establishes the TAMID Quant brand
  (e.g., social avatars, favicons, slide corners).
  - SVG: vector, scales infinitely
  - PNG: 64/128/256/600/1200 px wide, transparent background
  - favicon-16/32/48.png: small versions for browser tabs
  Colors:
    white  — for dark backgrounds (primary digital)
    blue   — accent / hero contexts
    gray   — secondary, low-contrast use
    black  — for white/light backgrounds (print)

02_horizontal_lockup/
  Mark on left, TAMID on top right, QUANT below right, blue rule.
  Standard signature lockup. Use in headers, footers, document covers.
  - 600/1200/2400 px wide, transparent background
  Variants: primary-dark, primary-light, mono-white, mono-black

03_stacked_lockup/
  Mark on top, wordmark below, with NEW YORK UNIVERSITY tag.
  Use when layout is square or portrait (social profiles, posters).
  - 800/1600 px wide, transparent background
  Same four color variants as horizontal.

04_wordmark_only/
  Just the typographic wordmark, no symbol. Use when the mark would be
  redundant or compete with another visual.
  - 500/1000/2000 px wide, transparent background
  Same four color variants.

05_inline_lockup/
  Single-line lockup: mark + TAMID + QUANT all on one line.
  Use for banner formats, email signatures, narrow horizontal bars.
  - 600/1200/2400 px wide
  Variants:
    primary-dark, primary-light, mono-white, mono-black (transparent)
    on-black, on-white, on-blue (solid backgrounds, ready to drop in)

07_stacked_singleline/
  Mark on top, TAMID QUANT on a single line below, with NEW YORK UNIVERSITY tag.
  Use when you want the vertical stacked feel but with the wordmark all on one row.
  Good for square-ish layouts that aren't tall enough for the two-line stacked version.
  - 800/1600 px wide
  Variants:
    primary-dark, primary-light, mono-white, mono-black (transparent)
    on-black, on-white, on-blue (solid backgrounds)

06_with_backgrounds/
  Pre-composed versions on solid backgrounds. Drop straight into any
  context that needs a self-contained logo image.
  - badge-mark-*       : square badges (256/512/1024 px) for app icons,
                          avatars, profile pictures. 5 color combos.
  - horizontal-*       : full horizontal lockup on black/white/blue
                          (600/1200/2400 px)
  - stacked-*          : stacked lockup on black/white/blue (800/1600 px)
  - social-card-*      : 1:1 square cards for Instagram/LinkedIn posts
                          (600/1200 px)
  - inline-*           : single-line lockup on black/white/blue
                          (1200/2400 px)

USAGE RULES
-----------
- Never recolor outside the four-color palette
- Never fill the mark interior — it stays hollow
- Never apply effects (shadows, glows, gradients)
- Never stretch or distort proportions
- Never place primary-dark variants on light backgrounds (illegible)
- On the on-blue background, the rule under QUANT is white (not blue)
- Maintain clear space equal to the mark's cap-height on all sides
- Minimum sizes: digital 80px wide, print 0.65 inch wide

TYPOGRAPHY
----------
Wordmarks use IBM Plex Mono Bold. The full type system uses
IBM Plex Mono in five weights (Light, Regular, Medium, SemiBold, Bold).
See the brand book for full type specifications.
